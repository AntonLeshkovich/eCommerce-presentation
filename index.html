About eCommerce Application

The project is developed using React. In the eCommerce Application project, we created an application for purchasing goods using CommerceTools (to create an API client). During the development of the "E-Commerce Application", we set up a development environment. The following pages were created: registration, profile login, product categories and subcategories, product, shopping cart, “About us”, profile, home page.

Stack

  • React
  • TypeScript
  • CSS
  • REST API
  • axios
  • Prettier
  • Husky
  • Jest
  • Eslint
  • CommerceTools

Install and usage project

  1. Clone the repository 'git clone https://github.com/Rigelion154/eCommerce-Application.git'
  2. Сhange directory 'cd eCommerce-Application'
  3. Change branch 'git checkout develop'
  4. Install NPM packages 'npm install'
  5. Run ESLint 'npm run lint'
  6. Run Prettier 'npm run format'
  7. Run tests 'npm run test'
  8. Run Husky 'npm run prepare'
  9. Usage 'npm run start'
